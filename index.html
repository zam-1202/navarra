About

Navarra is a local surname from Spain. It is derived from the Basque word "nava," or "naba," which means "plain near the mountains." This is my middle name and I chose it to represent my maternal family rather than my original surname 'Norte'.

The purpose of this website is to display the element of art and the how we utilize the basic web code in Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS).
